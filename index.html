SheetDiff

SheetDiff is a Google Sheets™ add-on for comparing two tabs in the currently open spreadsheet, reviewing differences, and safely applying approved updates back to source tabs.

What SheetDiff Does

  • Compare two tabs in the same spreadsheet using one or more key columns.
  • Create report, side-by-side, duplicate review, and merge review tabs inside that spreadsheet.
  • Highlight differences on the source tabs and apply approved write-backs after confirmation.
  • Store limited app settings such as presets, history, highlight context, and license state.

How Google User Data Is Used

  • SheetDiff accesses only the spreadsheet data in the file the user currently has open.
  • Spreadsheet row and cell data is processed inside Google Sheets™ / Apps Script to generate comparison and review outputs.
  • Spreadsheet content is not sent to third parties. External requests are used only for Gumroad license validation.
