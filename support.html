Common Issues

Why does SheetDiff block my comparison?
Every compared row must have a non-blank value in the selected key column or key columns.

Why can I not compare another spreadsheet?
Cross-file comparison requires Pro and the target spreadsheet must be accessible to your Google account.

Why are scheduled summaries not running?
Scheduled summaries require Pro and must be tested on a real versioned Add-on deployment because editor add-on test deployments do not support installable triggers.

Is it safe to use merge actions?
Merge actions write changes back into the source sheets. Review the merge review sheet carefully before confirming.

What to Include in a Support Request

Include the spreadsheet URL, the sheet names involved, the selected key columns, the exact action you ran, and a screenshot of the error if possible.

Data Deletion Requests

To request deletion of app-associated stored data, email sameer.mylavarapu21@gmail.com from the same Google account you used with SheetDiff and use the subject "SheetDiff data deletion request." Include the spreadsheet URL if your request relates to generated report or review tabs.
