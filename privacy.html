SheetDiff Privacy Policy

Last updated: March 28, 2026

SheetDiff is a Google Sheets™ add-on that compares spreadsheet tabs, highlights differences, creates review sheets, and helps users reconcile approved changes. This policy explains what Google user data SheetDiff accesses, how that data is used, how it is stored and protected, whether it is shared, and how users can request deletion.

Data Accessed

SheetDiff accesses only the Google Sheets™ data in the currently open spreadsheet that a user explicitly chooses to work with. This can include sheet names, selected ranges, row and cell values, and spreadsheet structure needed to compare two tabs, generate report tabs, create side-by-side views, create duplicate review sheets, create merge review sheets, highlight rows, and apply approved changes back to source sheets.

If a user activates a paid license, SheetDiff accesses the license key the user submits for verification.

Data Usage

SheetDiff uses Google user data only to deliver the app's requested functionality. Spreadsheet data is processed to compare sheets, identify added, removed, changed, or duplicate rows, generate report and review tabs, highlight differences, prepare merge decisions, and apply approved write-backs when the user confirms those actions.

License keys are used only to validate or revalidate access to paid features. SheetDiff does not use Google user data for advertising and does not use Google user data to train generalized AI or machine learning models.

Data Storage and Protection

SheetDiff does not operate its own external database for spreadsheet comparison data. Spreadsheet row and cell data stays in the user's Google Sheets™ files and Google Apps Script execution environment. Generated report and review tabs are created inside the user's spreadsheet.

SheetDiff stores limited app-associated state in Google Apps Script PropertiesService. This can include license status, trial status, daily usage counts, saved presets, comparison history, and highlight context. Access to this data is limited to the Apps Script project running the add-on under the permissions granted by the user.

Data Sharing

SheetDiff does not sell Google user data and does not share spreadsheet row or cell data with third parties. Spreadsheet comparison data remains inside Google Sheets™ and Google Apps Script.

The only third-party sharing performed by SheetDiff is for license verification with Gumroad. When a user activates or revalidates a license, SheetDiff sends the submitted license key and the Gumroad product identifier to Gumroad's license verification endpoint. Spreadsheet row and cell data are not sent to Gumroad.

Data Retention and Deletion

Generated report and review tabs remain in the user's spreadsheet until the user deletes them or uses SheetDiff cleanup actions. App-associated state stored in Apps Script PropertiesService remains until it is replaced, removed by the app, or deleted in response to a user request.

Users may request deletion of app-associated stored data by emailing sameer.mylavarapu21@gmail.com from the same Google account they used with SheetDiff and including SheetDiff data deletion request in the subject line. Upon request, we will delete the stored app-associated data that can be removed from the Apps Script project for that user. Users can also delete generated sheets directly from their own spreadsheet files at any time.

Contact

For privacy questions or deletion requests, contact sameer.mylavarapu21@gmail.com.
