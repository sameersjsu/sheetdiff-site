SheetDiff Privacy Policy

Last updated: March 11, 2026

SheetDiff is a Google Sheets add-on that compares spreadsheet tabs, highlights differences, and helps users review merge decisions. This page describes what data SheetDiff processes and where that data goes.

What SheetDiff Accesses

SheetDiff accesses spreadsheet data that the user explicitly chooses to compare inside Google Sheets. The add-on reads selected sheets and may create generated report tabs, side-by-side views, duplicate review sheets, merge review sheets, and highlight status columns.

What SheetDiff Stores

SheetDiff stores limited user-level settings in Google Apps Script PropertiesService. This can include license status, trial status, daily usage counts, saved presets, saved schedules, comparison history, and highlight context.

External Services

SheetDiff sends license verification requests to Gumroad when a user activates or re-validates a license key. Those requests contain the Gumroad product ID and the submitted license key. SheetDiff does not send spreadsheet row data to Gumroad.

Email Delivery

If a user enables scheduled summaries, SheetDiff sends summary emails through Google Mail to the email address the user provided for that schedule.

Data Sharing

SheetDiff does not sell spreadsheet data or share spreadsheet row data with third parties. Spreadsheet comparison data remains inside Google Sheets / Google Apps Script, except for the Gumroad license verification requests described above and user-requested email summaries.

Data Retention

User settings stored in Apps Script properties remain until the user clears them, the add-on clears them, or the script project is removed. Generated report sheets remain in the spreadsheet until the user deletes them or uses SheetDiff's cleanup actions.
